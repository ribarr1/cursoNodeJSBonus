Bonus tarea curso Node JS

	Para ejecutar esta parte de la actividad 1 se debe tomar en cuenta que se tienen los siguientes parametros: 

		nombre  (nombre del alumno)
		alias: 'n'
	
		curso (id del curso) 
		alias:'c'
	
		cedula (cedula del alumno)
		alias:'d'

	Los 3 parametros son obligatorios.

	el comando sugerido para la ejecucion es:

		  node inscripcion inscribir -c=IdDelCurso -n=NombreAlumno  -d=CedulaAlumno
		  

    Luego ver el resultado por tu navegador en la ruta (localhost:8081)
	
	Si solo quieres ver los cursos disponibles ejecutas:
	
	node mostrarCursos
	
	    Luego ver el resultado por tu navegador en la ruta (localhost:8082)
